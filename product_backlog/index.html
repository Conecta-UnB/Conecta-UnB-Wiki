    Histórias de usuário

    Feature US Critérios de aceitação História de usuário Prioridade
    F01 US01 - O usuário deve poder enviar matrícula, nome, e-mail, telefone e senha ao sistema
    - O usuário deve ser informado se o cadastro foi realizado com sucesso
    		Eu como visitante desejo realizar cadastro no site para poder ter acesso com privilégio must have
    F01 US02 - O usuário participante deve poder enviar uma solicitação de acesso com privilégio de gestor
    - O usuário gestor deve poder aceitar uma solicitação de acesso com privilégio    		 Eu como participante desejo solicitar acesso como gestor para criar publicações e/ou tópicos de fórum should have
    F02 US03 - O usuário deve poder informar e-mail e senha
    - O usuário deve poder acessar o sistema estando conectado    		 Eu como administrador/gestor/participante desejo logar no meu perfil com o objetivo de realizar alguma ação que requer autenticação must have
    F02 US04 - O usuário deve poder solicitar recuperação de senha
    - O usuário deve receber um e-mail com as instruções pra redefinir senha
    - O usuário deve poder indicar a nova senha    		 Eu como administrador/gestor/participante desejo resgatar minha senha com o objetivo de fazer login could have
    F03 US05 - O usuário deve ter acesso a um formulário de criação de notícia
    - O usuário deve preencher os campos de título, conteúdo, autor e imagem (opcional)
    - O usuário deve poder enviar o formulário    		 Eu como administrador/gestor desejo criar uma notícia com o objetivo de publicar novas informações must have
    F03 US06 - O usuário deve poder selecionar a notícia a ser excluída
    - O usuário deve poder apagar a notícia    		 Eu como administrador desejo excluir uma notícia com o objetivo de remover a informação should have
    F04 US07 - O usuário deve poder colocar tag para filtragem Eu como administrador/gestor desejo selecionar filtros (data, palavra-chave) para a publicação com o objetivo de facilitar acesso para os usuários could have
    F04 US08 - O usuário deve poder selecionar filtro (tag ou texto) para buscar notícias e eventos
    - O usuário deve poder visualizar as notícias e eventos relacionadas ao filtro    		 Eu como visitante desejo filtrar as publicações com o intuito de visualizar publicações de caráter específico could have
    F05 US09 - O usuário deve poder acessar uma página com as notícias
    - O usuário deve poder acessar uma notícia específica selecionada    		 Eu como visitante desejo visualizar notícias com o objetivo de me informar must have
    F05 US10 - O usuário deve poder enviar um formulário de denúncia ao sistema Eu como visitante desejo denunciar uma notícia com o objetivo de alertar os administradores sobre seu conteúdo should have
    F06 US11 - O usuário deve ter acesso a um formulário de criação de evento
    - O usuário deve preencher os campos de título, descrição, organizador e imagem (opcional)
    - O usuário deve poder enviar o formulário    		 Eu como administrador/gestor desejo publicar um evento com o objetivo de manter as pessoas a par de novos eventos must have
    F06 US12 - O usuário deve poder selecionar o evento a ser excluída
    - O usuário deve poder apagar o evento    		 Eu como administrador/gestor desejo excluir um evento com o objetivo de retirar da página could have
    F06 US13 - O usuário deve poder ter acesso a um formulário para editar qualquer um dos dados
    - O usuário deve poder enviar os dados atualizados ao sistema    		 Eu como administrador/gestor desejo editar dados de um evento com o objetivo de manter a publicação atualizada should have
    F07 US14 - O usuário deve poder acessar uma página com os eventos
    - O usuário deve poder acessar um evento específico selecionado    		 Eu como visitante desejo visualizar um evento com o objetivo de acessar as informações sobre o mesmo must have
    F08 US15 - O usuário deve poder acessar o mapa
    - O usuário deve poder visualizar locais do campi    		 Eu como visitante desejo visualizar o mapa da FGA com o objetivo de me localizar could have
    F08 US16 - O usuário deve poder ter acesso as fotos
    - O usuário deve poder visualizar as fotos    		 Eu como visitante desejo acessar uma galeria de fotos da FGA com o objetivo de conhecer o espaço do campi could have
    F09 US17 - O usuário deve ter acesso a um formulário de registro de EJ/EC/Lab
    - O usuário deve preencher os campos de nome, descrição, e-mail, telefone, facebook, instagram, site e foto (opcional)
    - O usuário deve poder enviar o formulário    		 Eu como administrador/gestor desejo registrar uma empresa júnior/equipe de competição/laboratório com o objetivo de expô-la no site must have
    F09 US18 - O usuário deve poder ter acesso a um formulário para editar qualquer um dos dados
    - O usuário deve poder enviar os dados atualizados ao sistema    		 Eu como administrador/gestor desejo atualizar os dados de uma EJ/EC/Lab com o objetivo de manter o registro e informações atualizados must have
    F10 US19 - O usuário deve poder acessar uma página com as EJs/ECs/Labs
    - O usuário deve poder acessar uma EJ/EC/Lab específica selecionada    		 Eu como visitante desejo ter acesso aos dados das EJs/ECs/Lab com o intuito de me informar sobre a equipe must have
    F11 US20 - O usuário deve ter acesso a um formulário de criação de tópico de discussão
    - O usuário deve preencher os campos de título e descrição
    - O usuário deve poder enviar o formulário    		 Eu como administrador/gestor desejo criar um tópico de discussão com o objetivo de promover interação sobre um assunto must have
    F11 US21 - O usuário deve poder selecionar o tópico de discussão a ser excluído
    - O usuário deve poder apagar o tópico de discussão    		 Eu como administrador desejo deletar um tópico de discussão com o objetivo de desfazer/encerrar o debate should have
    F12 US22 - O usuário deve poder enviar um formulário de denúncia ao sistema Eu como visitante desejo denunciar um tópico/comentário do fórum com o objetivo de solicitar revisão dos administradores should have
    F12 US23 - O usuário deve ter acesso a um formulário de criação de tópico de discussão
    - O usuário deve preencher o campo de descrição
    - O usuário deve poder enviar o formulário    		 Eu como participante desejo comentar num tópico de discussão com o objetivo de interagir num debate must have
